Welcome to the FastAPI Image Prediction Application

Using the API Directly

For programmatic access, you can interact with the API directly:

  1. Upload Image:

    Send a POST request with your image to the /images/ endpoint.

  2. List Models:

    Send a GET request to the /models/ endpoint to view the list of available models.

  3. Predict:

    Send a POST request with the model name to the /predict/ endpoint to perform a prediction.

Using the Web UI

You can also interact with the application via a user-friendly web interface:

  1. Upload Image:

    Visit the Upload Image page to upload an image through the web interface.

  2. List Models:

    Go to the List Models page to see available models and select one for prediction.

  3. Predict:

    After selecting a model, use the Predict page to perform a prediction.
